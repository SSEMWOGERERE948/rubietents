# Rubie Tents Company Limited Website

A clean Next.js landing website for Rubie Tents Company Limited.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn-style UI components
- lucide-react icons

## How to run

```bash
npm install
npm run dev
```

Open:

```bash
http://localhost:3000
```

## Important edits

Replace these placeholders:

- `+256 XXX XXX XXX`
- `info@rubietents.com`
- `https://wa.me/256XXXXXXXXX`
- Kampala location details
- Gallery placeholder blocks with real photos
- Logo placeholder `RT` with the actual Rubie Tents logo

## Logo colors

Update `tailwind.config.ts`:

```ts
brand: {
  DEFAULT: "#8B1E3F",
  dark: "#5F132A",
  gold: "#D99A2B",
  sand: "#FFF7EA",
  ink: "#191716",
  muted: "#6F625B",
}
```

Use the exact colors from the Rubie Tents logo when available.
